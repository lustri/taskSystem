Sistema de tasks

O sistema de tasks é uma aplicação web CRUD para listagem, criação, remoção e leitura de tasks.
Para o desenvolvimento do mesmo foi utilizado um banco MySQL e linguagem PHP.

----------------------------------------------------------------------------

Execução:

1) Executar o software WampServer (ou similar) para criar um servidor local

2) Transferir o projeto para dentro da pasta do servidor local (No caso do WampServer, dentro da pasta "www")

3) Executar os comandos do arquivo "mySQLCommands" para a criação do banco e das tabelas

4) Editar, se necessário, as configurações da conexão com o banco no arquivo "connection.php"

5) Acessar a aplicação através de um browser
